# Audio Visualizer BE

Back end for audio visualizer

## Description

This backend is needed for audiovisualizer-FE to properly record and download visualizations

## :rocket: How to start

### Instaling dependencies

```
yarn install
```

### Starting in dev mode

```
yarn start:dev
```

### Starting aplication

```
yarn start
```
